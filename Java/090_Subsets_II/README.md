 </p>
 <p>
  Given a collection of integers that might contain duplicates,
  <b>
   <i>
    nums
   </i>
  </b>
  , return all possible subsets.
 </p>
 <p>
  <b>
   Note:
  </b>
  <br/>
 </p>
 <ul>
  <li>
   Elements in a subset must be in non-descending order.
  </li>
  <li>
   The solution set must not contain duplicate subsets.
  </li>
 </ul>
 <p>
  For example,
  <br/>
  If
  <b>
   <i>
    nums
   </i>
  </b>
  =
  <code>
   [1,2,2]
  </code>
  , a solution is:
 </p>
 <pre>
[
  [2],
  [1],
  [1,2,2],
  [2,2],
  [1,2],
  []
]
</pre>
</div>
