 <p>
  Given a 2D board containing
  <code>
   'X'
  </code>

  <code>
   'O'
  </code>

  <code>
   'X'
  </code>
  .
 </p>
 <p>
  A region is captured by flipping all
  <code>
   'O'
  </code>
  s into
  <code>
   'X'
  </code>
  s in that surrounded region.
 </p>
 <p>
  For example,
  <br/>
 </p>
 <pre>
X X X X
X O O X
X X O X
X O X X
</pre>
 <p>
  After running your function, the board should be:
 </p>
 <pre>
X X X X
X X X X
X X X X
X O X X
</pre>
