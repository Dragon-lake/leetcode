# Combination Sum II

## Problem：

<div class="question-content">
 <p>
 </p>
 <p>
  Given a collection of candidate numbers (
  <b>
   <i>
    C
   </i>
  </b>
  ) and a target number (
  <b>
   <i>
    T
   </i>
  </b>
  ), find all unique combinations in
  <b>
   <i>
    C
   </i>
  </b>
  where the candidate numbers sums to
  <b>
   <i>
    T
   </i>
  </b>
  .
 </p>
 <p>
  Each number in
  <b>
   <i>
    C
   </i>
  </b>
  may only be used
  <b>
   once
  </b>
  in the combination.
 </p>
 <p>
  <b>
   Note:
  </b>
  <br/>
 </p>
 <ul>
  <li>
   All numbers (including target) will be positive integers.
  </li>
  <li>
   Elements in a combination (
   <i>
    a
   </i>
   <sub>
    1
   </sub>
   ,
   <i>
    a
   </i>
   <sub>
    2
   </sub>
   , … ,
   <i>
    a
   </i>
   <sub>
    k
   </sub>
   ) must be in non-descending order. (ie,
   <i>
    a
   </i>
   <sub>
    1
   </sub>
   ≤
   <i>
    a
   </i>
   <sub>
    2
   </sub>
   ≤ … ≤
   <i>
    a
   </i>
   <sub>
    k
   </sub>
   ).
  </li>
  <li>
   The solution set must not contain duplicate combinations.
  </li>
 </ul>
 <p>
  For example, given candidate set
  <code>
   10,1,2,7,6,1,5
  </code>
  and target
  <code>
   8
  </code>
  ,
  <br/>
  A solution set is:
  <br/>
  <code>
   [1, 7]
  </code>
  <br/>
  <code>
   [1, 2, 5]
  </code>
  <br/>
  <code>
   [2, 6]
  </code>
  <br/>
  <code>
   [1, 1, 6]
  </code>
  <br/>
 </p>
</div>
