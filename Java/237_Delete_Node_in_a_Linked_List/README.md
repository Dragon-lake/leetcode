 </p>
 <p>
  Supposed the linked list is
  <code>
   1 -&gt; 2 -&gt; 3 -&gt; 4

  <code>
   3
  </code>
  , the linked list should become
  <code>
   1 -&gt; 2 -&gt; 4
  </code>
  after calling your function.
 </p>
