 <p>
  Given a binary tree containing digits from
  <code>
   0-9
  </code>

 </p>
 <p>

  </code>

  <code>
   123
  </code>
  .
 </p>
 <p>
  Find the total sum of all root-to-leaf numbers.
 </p>
 <p>
  For example,
 </p>
 <pre>
    1
   / \
  2   3
</pre>
 <p>
  The root-to-leaf path
  <code>
   1-&gt;2
  </code>
  represents the number
  <code>
   12
  </code>
  .
  <br/>
  The root-to-leaf path
  <code>
   1-&gt;3
  </code>
  represents the number
  <code>
   13
  </code>
  .
 </p>
 <p>
  Return the sum = 12 + 13 =
  <code>
   25
  </code>
  .
 </p>
