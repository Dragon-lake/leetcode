 <p>
  Given a string that contains only digits
  <code>
   0-9
  </code>
  and a target value, return all possibilities to add
  <b>
   binary
  </b>
  operators (not unary)
  <code>
   +
  </code>
  ,
  <code>
   -
  </code>
  , or
  <code>
   *
  </code>
  between the digits so they evaluate to the target value.
 </p>
 <p>
  Examples:
  <br/>
 </p>
 <pre>"123", 6 -&gt; ["1+2+3", "1*2*3"] 
"232", 8 -&gt; ["2*3+2", "2+3*2"]
"105", 5 -&gt; ["1*0+5","10-5"]
"00", 0 -&gt; ["0+0", "0-0", "0*0"]
"3456237490", 9191 -&gt; []
</pre>
