 <p>
  Given a positive integer
  <i>
   n
  </i>
  , find the least number of perfect square numbers (for example,
  <code>
   1, 4, 9, 16, ...
  </code>
  ) which sum to
  <i>
   n
  </i>
  .
 </p>
 <p>
  For example, given
  <i>
   n
  </i>
  =
  <code>
   12
  </code>
  , return
  <code>
   3
  </code>
  because
  <code>
   12 = 4 + 4 + 4
  </code>
  ; given
  <i>
   n
  </i>
  =
  <code>
   13
  </code>
  , return
  <code>
   2
  </code>
  because
  <code>
   13 = 4 + 9
  </code>
  .
 </p>
