# Word Pattern

## Problemï¼š

<div class="question-content">
 <p>
 </p>
 <p>
  Given a
  <code>
   pattern
  </code>
  and a string
  <code>
   str
  </code>
  , find if
  <code>
   str
  </code>
  follows the same pattern.
 </p>
 <p>
  Here
  <b>
   follow
  </b>
  means a full match, such that there is a bijection between a letter in
  <code>
   pattern
  </code>
  and a
  <b>
   non-empty
  </b>
  word in
  <code>
   str
  </code>
  .
 </p>
 <p>
  <b>
   Examples:
  </b>
  <br/>
 </p>
 <ol>
  <li>
   pattern =
   <code>
    "abba"
   </code>
   , str =
   <code>
    "dog cat cat dog"
   </code>
   should return true.
  </li>
  <li>
   pattern =
   <code>
    "abba"
   </code>
   , str =
   <code>
    "dog cat cat fish"
   </code>
   should return false.
  </li>
  <li>
   pattern =
   <code>
    "aaaa"
   </code>
   , str =
   <code>
    "dog cat cat dog"
   </code>
   should return false.
  </li>
  <li>
   pattern =
   <code>
    "abba"
   </code>
   , str =
   <code>
    "dog dog dog dog"
   </code>
   should return false.
  </li>
 </ol>
 <p>
  <b>
   Notes:
  </b>
  <br/>
  You may assume
  <code>
   pattern
  </code>
  contains only lowercase letters, and
  <code>
   str
  </code>
  contains lowercase letters separated by a single space.
 </p>
