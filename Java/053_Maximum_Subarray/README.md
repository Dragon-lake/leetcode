# Maximum Subarray

## Problem：

<div class="question-content">
 <p>
 </p>
 <p>
  Find the contiguous subarray within an array (containing at least one number) which has the largest sum.
 </p>
 <p>
  For example, given the array
  <code>
   [−2,1,−3,4,−1,2,1,−5,4]
  </code>
  ,
  <br/>
  the contiguous subarray
  <code>
   [4,−1,2,1]
  </code>
  has the largest sum =
  <code>
   6
  </code>
  .
 </p>
